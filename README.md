# Yen-Nan (Aaron) Ho

**AI Engineer / Data Scientist** based in Taiwan.  
I work on applied AI systems that connect **field data**, **signal analysis**, **backend platforms**, **diagnostic reporting**, and **human-in-the-loop AI workflows**.

My recent work focuses on industrial AI and predictive maintenance: vibration monitoring, electrical-signal analysis, remaining-life estimation, automated diagnostic reports, and reliability-oriented platform engineering. Public descriptions are intentionally high-level because some work involves confidential customer data and proprietary systems.

## What I Work On

- **Industrial AI / Predictive Maintenance**: vibration signals, PRPD / electrical-discharge analysis, anomaly detection, trend and RUL interpretation.
- **AI Engineering**: report-generation workflows, human review gates, deterministic fallback, prompt constraints, and agent-assisted engineering loops.
- **Backend / Platform Engineering**: Python, Django, PostgreSQL, Redis, asynchronous workflows, data migration, release packaging, and system integration.
- **Data Science / Research**: time-series modeling, dynamic embedding, NLP, deep learning, web crawling, GIS analysis, and bioinformatics.

## Selected Professional Work

- Built and shipped internal industrial AI platform features for equipment monitoring, diagnostic reporting, and predictive-maintenance workflows.
- Processed large-scale confidential field datasets for industrial signal analysis, including multi-sensor PRPD and vibration cases.
- Improved a large diagnostic workload from **35-42 GiB peak memory to 822 MiB**, turning a failed long-running task into a successful run.
- Optimized batch waveform analysis from roughly **35 hours to under 1 hour** for **25,000+ files**.
- Built human + AI workflows where AI accelerates drafting and analysis, while human review controls assumptions, evidence, and customer-facing conclusions.
